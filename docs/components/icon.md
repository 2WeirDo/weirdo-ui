# Icon图标

对`fontawesome`图标库进行封装，涵盖了各种常见的图标需求。

## 基础用法

传入`icon`属性设置要使用的Icon图标。

<preview path="../demo/Icon/Basic.vue"></preview>

## 扩展用法

Icon组件是对`fontawesome`图标库的封装，因此该图标库的相关属性均可以设置, 比如颜色`color`、大小`size`, 旋转角度`rotation`。
