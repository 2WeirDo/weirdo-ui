# weirdo-ui

仿 Element-Plus 组件库开发框架

**weirdo-ui**是一个 仿照 `Element-Plus` 实现的一个简易版的**Vue3**组件库, 技术栈为 `Vue3、TypeScript、Vite、Vitest、VitePress、Rollup、Postcss`。

## 下载项目

```sh
git clone https://github.com/2WeirDo/weirdo-ui.git
``` 

## 快速启动

```sh
pnpm install
```

## 启动项目

```sh
pnpm dev
```

## 启动 vitepress 文档

```sh
pnpm pnpm docs:dev
```
